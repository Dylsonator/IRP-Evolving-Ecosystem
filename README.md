IRP Evolving Ecosystem

How to use the system
===

# Open the Unity scene and press Play. The ecosystem will spawn fish, plants and other resources automatically.

# Each fish acts on its own using its genome and neural brain. They will move around, eat, hunt, flee, mate or rest depending on their traits and the current environment.

# The simulation runs in generations. After each generation, the system evaluates the fish, selects successful or useful behaviours, mutates them, and spawns the next population.

# Fish can be selected with the camera controls to view their current behaviour, health, energy, diet type and role. The CSV files are also saved automatically so the results can be used for graphs and evaluation.

# A saved evolved population can be loaded later to show how the ecosystem behaves after several generations of adaptation. 

# saved data can be located at C:"User":Appdata:Locallow:DefaultCompany:IRP Evolving Ecosystems
